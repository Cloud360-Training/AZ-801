# Hands-On Labs & Implementation Exercises

Practical experience in implementing Windows Server security is essential for the AZ-801 exam.

## 🧪 Microsoft Learn Sandbox

The Microsoft Learn Sandbox provides a free, temporary Azure environment where you can implement security solutions.

### How to Access the Sandbox
1. Navigate to any Microsoft Learn module that includes a hands-on exercise.
2. Sign in with your Microsoft account.
3. Click the **Activate sandbox** button.
4. Review the permissions and accept.
5. You will have a set amount of time to use the sandbox environment.

## 📝 Recommended Implementation Exercises

### Lab 1: Implement Windows Server Security
* **Objective**: Configure Exploit Protection, WDAC, and Credential Guard.
* **Microsoft Learn Guide**: [Secure Windows Server OS](https://learn.microsoft.com/en-us/training/modules/secure-windows-server-operating-system/)
* **Key Skills**: Exploit Protection, WDAC policies, Credential Guard configuration.

### Lab 2: Secure Active Directory
* **Objective**: Implement password policies, LAPS, and Defender for Identity.
* **Microsoft Learn Guide**: [Secure AD DS](https://learn.microsoft.com/en-us/training/modules/secure-hybrid-active-directory-infrastructure/)
* **Key Skills**: Password policy configuration, LAPS deployment, Defender for Identity setup.

### Lab 3: Implement Network Security
* **Objective**: Configure Windows Defender Firewall and NSGs.
* **Microsoft Learn Guide**: [Secure networking](https://learn.microsoft.com/en-us/training/modules/manage-windows-defender-firewall/)
* **Key Skills**: Firewall rules, connection security, NSG configuration.

### Lab 4: Configure Hybrid Management
* **Objective**: Set up Defender for Cloud and Sentinel integration.
* **Microsoft Learn Guide**: [Implement hybrid management](https://learn.microsoft.com/en-us/training/modules/manage-security-defender-for-cloud/)
* **Key Skills**: Defender for Cloud setup, Sentinel ingestion, security monitoring.
