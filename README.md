# AZ-801 Microsoft Windows Server Hybrid Administrator Associate: Ninja Training

Welcome to the **AZ-801 Microsoft Windows Server Hybrid Administrator Associate Ninja Training**! This repository is your comprehensive resource for preparing for the AZ-801 certification exam, Microsoft's advanced certification for Windows Server administrators.

**Maintained by [Cloud360 Training](https://cloud360.co)** - Your trusted partner for Azure certification preparation.

Following the Microsoft Ninja Training format, this guide takes you from security fundamentals to advanced hybrid management through a carefully structured learning path.

## 🎯 Training Philosophy

This training program prioritizes resources in the following order to ensure you get the most accurate and effective preparation:
1. **Microsoft Official Documentation & Learn Modules** (Highest Priority)
2. **Cloud360 Training Videos & Resources**
3. **Hands-on Labs & Sandbox Exercises**
4. **Community Resources & Practice Tests**

## 📚 Course Structure

The training is organized into four modules that directly map to the official AZ-801 exam skills measured.

### [Module 1: Secure Windows Server Operating System (25-30%)](./1-WINDOWS-SECURITY/README.md)
* Configure and manage Exploit Protection
* Configure Windows Defender Application Control
* Configure Windows Defender Credential Guard
* Implement operating system security

### [Module 2: Secure a Hybrid Active Directory Infrastructure (25-30%)](./2-AD-SECURITY/README.md)
* Configure password policies
* Implement Microsoft Entra Password Protection
* Manage protected users
* Harden domain controllers
* Implement Microsoft Defender for Identity

### [Module 3: Secure Windows Server Networking (20-25%)](./3-NETWORK-SECURITY/README.md)
* Manage Windows Defender Firewall
* Implement domain isolation
* Implement connection security rules
* Create and configure network security groups

### [Module 4: Implement Hybrid Management and Monitoring (20-25%)](./4-HYBRID-MANAGEMENT/README.md)
* Manage security using Microsoft Defender for Cloud
* Manage security using Microsoft Defender for Servers
* Implement ingestion into Microsoft Sentinel
* Monitor and manage Windows Server security

## 🎓 Prerequisites

Before starting AZ-801, you should have:
* Passed the AZ-800 (Windows Server Hybrid Core Infrastructure Administrator) exam, or have equivalent experience
* Strong understanding of Windows Server security concepts
* Experience with Active Directory administration
* Familiarity with Azure security services
